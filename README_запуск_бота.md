# Запуск Telegram-бота «Консьерж Даши» — 5 минут, бесплатно

## Шаг 1. Получить токен у @BotFather
1. Открой Telegram, в поиске найди **@BotFather** (синяя галочка).
2. Нажми **Start**, отправь команду `/newbot`.
3. Придумай имя бота (например «Консьерж Даши») и username (должен кончаться на `bot`, напр. `dasha_concierge_bot`).
4. BotFather пришлёт строку вида `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` — **это твой BOT_TOKEN.** Скопируй.

## Шаг 2. Запустить
В Терминале (никакой установки пакетов не нужно — только стандартный Python):
```bash
cd "/Users/arina/Documents/Claude/Projects/Smart cat/slice"
export BOT_TOKEN="сюда_токен_от_BotFather"
python3 telegram_bot.py
```
Увидишь «Бот запущен».

> Для соло-демо `DASHA_CHAT_ID` задавать НЕ нужно: карточка Даши прилетит тебе же (ты и «клиент», и «Даша»). Если хочешь слать карточки в отдельный чат — узнай свой id у **@userinfobot** и добавь `export DASHA_CHAT_ID="твой_id"`.

## Шаг 4. Демо
1. Открой чат со своим ботом, нажми **Start**.
2. Отправь сырой DM клиента, например:
   > привет! нужен торт на др сына 18 июля, фанат динозавров, гостей человек 15, шоколадный
3. Тебе прилетит **карточка Даши** с ценой 7500 ₽ и кнопками [✅ Одобрить][✏️ Изменить][❌ Отклонить].
4. Попробуй ещё два:
   - «нужен торт на 25 человек на 4 июля, двухъярусный, с доставкой» → расчёт/или отказ, если неделя полная.
   - «крем был кислый, очень расстроена» → карточка «не автоматизирую, ответь лично».

## Что показывает демо
- Живой бесплатный автопилот в Telegram: DM → бриф → цена → карточка на аппрув в один тап.
- AI-суждение (тип сообщения, декор, тема) + детерминированная цена (`price_engine.py`).
- Дизайн под Дашу: отказ пишется за неё, чувствительное уходит человеку, всё через аппрув.

> Примечание: для бесплатного живого демо разбор DM сделан парсером. Место, где в прод встаёт настоящий LLM-агент (`agent_concierge.md`), помечено в коде как `[AI-HOOK]` — тогда бот поймёт любую формулировку, а не только шаблонные.
